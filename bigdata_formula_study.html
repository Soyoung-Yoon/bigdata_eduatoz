통계 기초 공식
이상값 검출 — ESD이상값
\(\mu - 3\sigma < \text{정상} < \mu + 3\sigma\)
μ: 평균, σ: 표준편차
이상값 검출 — IQR 방법이상값
\(Q_1 - 1.5\!\times\! IQR < \text{정상} < Q_3 + 1.5\!\times\! IQR\)
IQR = Q3 − Q1 (데이터 가운데 50%)
Min-Max 정규화스케일링
\(X' = \dfrac{X - X_{min}}{X_{max} - X_{min}}\)
결과: 0 ~ 1 범위 변환
Z-Score 표준화스케일링
\(Z' = \dfrac{X - \mu}{\sigma}\)
평균=0, 분산=1. 단위 없음(약분됨)
피어슨 상관계수상관
\(r = \dfrac{cov(x,y)}{\sigma_x \sigma_y}\)
범위: −1 ≤ r ≤ 1. 등간·비율척도. 이상값에 민감
표본 공분산상관
\(cov(x,y) = \dfrac{\sum(x_i-\bar{x})(y_i-\bar{y})}{n-1}\)
모집단: 분모 N / 표본: 분모 n−1
산술·조화·기하 평균평균
\(\bar{x}=\dfrac{\sum x_i}{n}\quad H=\dfrac{n}{\sum\frac{1}{x_i}}\quad G=\!\left(\prod x_i\right)^{1/n}\)
조화: 평균 속도 / 기하: 물가·성장률
표본 분산·표준편차·변동계수산포
\(s^2=\dfrac{\sum(x_i-\bar{x})^2}{n-1}\quad s=\sqrt{s^2}\quad CV=\dfrac{s}{\bar{x}}\)
분모 = n−1 (자유도). CV는 음수 없음
왜도별 대소 관계분포
\(\text{음의 왜도: } Mean \lt Median \lt Mode\)
\(\text{양의 왜도: } Mode \lt Median \lt Mean\)
정규분포: 왜도 = 첨도 = 0 (초과 첨도 기준)

확률 · 분포 공식
조건부 확률확률
\(P(A|B)=\dfrac{P(A\cap B)}{P(B)},\quad P(B)>0\)
~일 때(~중에서) ~일 확률
독립·배반·종속 사건확률
\(\text{독립: }P(A\cap B)=P(A)\cdot P(B)\)
\(\text{배반: }P(A\cap B)=0\)
\(\text{종속: }P(A\cap B)=P(A|B)\cdot P(B)\)
베이즈 정리베이즈
\(P(A|B)=\dfrac{P(B|A)\cdot P(A)}{P(B)}\)
전확률: \(P(B)=\sum P(B|A_i)\cdot P(A_i)\)
확률변수의 분산분포
\(Var(X)=E(X^2)-\{E(X)\}^2\)
\(\sigma(X)=\sqrt{Var(X)}\)
기댓값의 성질기댓값
\(E(aX\pm b)=aE(X)\pm b\)
\(E(aX\pm bY)=aE(X)\pm bE(Y)\)
분산·표준편차의 성질분산
\(Var(aX\pm b)=a^2 Var(X)\)
\(Var(aX\pm bY)=a^2Var(X)\pm b^2Var(Y)\)
\(\sigma(aX+b)=|a|\sigma(X)\)

이산형 확률분포 공식
베르누이 분포이산
\(E(X)=p\qquad Var(X)=pq\)
1회 시행, 성공(1)/실패(0). q=1−p
이항 분포이산
\(P(K\!=\!k)=\binom{n}{k}p^k(1\!-\!p)^{n-k}\)
\(E(X)=np\qquad Var(X)=np(1-p)\)
n번 독립 반복, 성공확률 일정
기하 분포이산
\(P(X\!=\!x)=(1\!-\!p)^{x-1}p\)
\(E(X)=\dfrac{1}{p}\qquad Var(X)=\dfrac{1-p}{p^2}\)
처음 성공이 나올 때까지 시도 횟수
포아송 분포이산
\(P(X\!=\!k)=\dfrac{e^{-\lambda}\lambda^k}{k!}\)
\(E(X)=\lambda\qquad Var(X)=\lambda\)
단위 시간/공간 발생 횟수. 모수 1개(λ)
이항분포 정규근사 (중심극한정리)근사
\(B(n,p)\xrightarrow{n\to\infty}N(np,\;npq)\)
n≥30이면 표본평균의 분포가 정규분포에 근사

표본분포 · 추정 · 검정통계량
표본평균의 분포표본분포
\(\bar{X}\sim N\!\left(\mu,\;\dfrac{\sigma^2}{n}\right)\quad SE=\dfrac{\sigma}{\sqrt{n}}\)
유한 모집단: \(SE=\sqrt{\dfrac{N-n}{N-1}}\cdot\dfrac{\sigma}{\sqrt{n}}\)
신뢰구간 · 오차한계추정
\(\bar{x}\pm z\cdot\dfrac{\sigma}{\sqrt{n}}\)
95%: \(z=1.96\quad\) 99%: \(z=2.576\)
오차한계 = 임계값 × 표준오차
Z-검정 통계량검정
\(Z=\dfrac{\bar{x}-\mu_0}{\sigma/\sqrt{n}}\)
모분산 알 때 또는 대표본(n≥30)
t-검정 통계량검정
\(t=\dfrac{\bar{x}-\mu_0}{s/\sqrt{n}}\)
모분산 모를 때, 소표본. 자유도 = n−1
카이제곱 · F 검정통계량검정
\(\chi^2=\dfrac{(n-1)s^2}{\sigma_0^2}\qquad F=\dfrac{s_1^2}{s_2^2}\)
χ²: 분산 1개 / F: 분산 2개 비교·ANOVA
카이제곱 검정통계량 (독립성·동질성)카이제곱
\(\chi^2=\sum\dfrac{(O_i-E_i)^2}{E_i}\quad E=\dfrac{\text{행합}\times\text{열합}}{\text{전체}}\)
자유도: (행수−1)×(열수−1)

회귀분석 · 규제 모형 공식
결정계수 R²회귀
\(R^2=\dfrac{SSR}{SST}=1-\dfrac{SSE}{SST}\)
1에 가까울수록 좋음. 독립변수 많아지면 커짐
MAE · MSE · RMSE회귀평가
\(MAE=\dfrac{\sum|y_i-\hat{y}_i|}{n}\quad MSE=\dfrac{\sum(y_i-\hat{y}_i)^2}{n}\)
\(RMSE=\sqrt{MSE}\)
0에 가까울수록 좋음. MAE: 이상치 영향 적음
MAPE · MSLE · RMSLE회귀평가
\(MAPE=\dfrac{100\%}{n}\sum\!\left|\dfrac{y_i-\hat{y}_i}{y_i}\right|\)
\(MSLE=\dfrac{\sum(\log(\hat{y}_i{+}1)-\log(y_i{+}1))^2}{n}\)
MSLE: 음수 불가(log 사용). 작게 예측에 큰 패널티
Lasso (L1 규제)규제
\(J=RSS+\lambda\sum_{j=1}^{p}|\beta_j|\)
절댓값(L1). 계수가 0이 될 수 있음 → 변수 선택
Ridge (L2 규제)규제
\(J=RSS+\lambda\sum_{j=1}^{p}\beta_j^2\)
제곱(L2). 계수가 0에 가까워지나 0 되지 않음
로지스틱 회귀 — Odds · Sigmoid분류
\(odds=\dfrac{p}{1-p}\quad logit=\log(odds)\)
\(P(Y=1)=\dfrac{1}{1+e^{-z}}\)
x₁이 1단위 증가 → odds는 e^β₁배 증가
Odds Ratio로지스틱
\(OR=\dfrac{a/b}{c/d}=\dfrac{ad}{bc}\)
OR > 1: odds 증가 / OR < 1: odds 감소
지니 지수 · 엔트로피 지수의사결정나무
\(Gini=1-\sum P_i^2\quad Entropy=-\sum P_i\log_2 P_i\)
CART: Gini / C5.0: Entropy / CHAID: χ²

── 확률 ──────────────────────────────────
확률조건부 확률\(P(A\cap B)/P(B)\)P(B)>0 조건
확률독립사건\(P(A\cap B)=P(A)\cdot P(B)\)서로 영향 없음
확률배반사건\(P(A\cap B)=0\)교집합=공집합
확률베이즈 정리\(P(B|A)\cdot P(A)/P(B)\)기저율 반영
기댓값E(aX±b)\(aE(X)\pm b\)선형성
분산Var(aX±b)\(a^2 Var(X)\)상수 b는 영향 없음
분산Var(aX±bY)\(a^2Var(X)\pm b^2Var(Y)\)독립일 때 성립
분산Var(X) 계산\(E(X^2)-\{E(X)\}^2\)핵심 계산 공식
── 이산형 확률분포 ───────────────────────
이산분포베르누이\(E=p,\quad Var=pq\)1회 시행
이산분포이항분포 E·Var\(E=np,\quad Var=np(1-p)\)n회 독립 반복
이산분포이항분포 PMF\(\binom{n}{k}p^k(1-p)^{n-k}\)k번 성공 확률
이산분포기하분포\(E=1/p,\quad Var=(1-p)/p^2\)첫 성공까지 횟수
이산분포포아송 PMF\(e^{-\lambda}\lambda^k/k!\)단위 시간/공간 발생 수
이산분포포아송 E·Var\(E=\lambda,\quad Var=\lambda\)모수 1개(λ)

분류평가정밀도(Precision)\(TP/(TP+FP)\)분모=예측 양성. "예Pre"
분류평가정확도(Accuracy)\((TP+TN)/(TP+TN+FP+FN)\)전체 중 올바른 예측
분류평가특이도(Specificity)\(TN/(TN+FP)\)FP Rate = 1 - Specificity
분류평가F1 Score\(2\cdot Prec\cdot Rec/(Prec+Rec)\)Precision·Recall 조화평균
분류평가FP Rate\(FP/(TN+FP)\)ROC 곡선 X축
── 거리 측도 · 연관분석 ──────────────────
거리유클리드\(\sqrt{\sum(x_i-y_i)^2}\)직선 거리
거리맨해튼\(\sum|x_i-y_i|\)격자 거리
거리민코프스키\((\sum|x_i-y_i|^p)^{1/p}\)유클리드·맨해튼의 일반형
거리코사인 유사도\(A\cdot B/(\|A\|\|B\|)\)두 벡터 사잇각, 텍스트 유사도
연관분석지지도\(P(A\cap B)\)동시 발생 비율
연관분석신뢰도\(P(A\cap B)/P(A)\)A 발생 시 B 발생 확률
연관분석향상도\(P(A\cap B)/(P(A)P(B))\)>1:양의 관계 / =1:무관 / <1:음의 관계
