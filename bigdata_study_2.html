중심경향 통계량: 평균·중앙값·최빈값

★★★★★
평균중앙값최빈값왜도
핵심 통계량▼
  • 평균(Mean) — 데이터의 합 / n. 이상값에 민감
  • 중앙값(Median) — 이상값에 강건함
  • 최빈값(Mode) — 범주형 변수에 사용
  • 음의 왜도: Mean < Median < Mode
  • 양의 왜도: Mean > Median > Mode

💡 평균의 종류

산술평균: 일반적인 평균
조화평균: N개 양수의 역수들의 산술평균의 역수 (평균 속도)
기하평균: N개 자료를 곱한 후 N 제곱근 계산 (물가 상승률, 경제 성장률)

산포(Dispersion) 통계량

★★★★★
분산IQR변동계수이상값
산포 통계량▼
  • 이상값에 민감: 평균, 분산, 표준편차, 범위(Range) — 최솟값과 최댓값을 포함하는 통계량
  • 이상값에 민감하지 않음: 중앙값, IQR — 최솟값과 최댓값을 포함하지 않는 통계량
  • 사분위수 범위(IQR) = Q3 - Q1 (Q1=25%, Q3=75%). 이상값 영향 없음
  • 변동계수 — 음수 없음
  • 표본 분산 = 편차 제곱의 합 / (n-1)
핵심 공식
분산 = 0 → 모든 값이 동일
평균 = 0 → 분산 = 각 값의 제곱 평균 (편차 = 0)

분포 통계량: 첨도·왜도

★★★★☆
첨도왜도정규분포
분포 특성▼
  • 정규분포의 경우: 첨도 = 왜도 = 0
  • 정규분포: 기댓값, 최빈값, 중앙값이 모두 평균과 같음
  • 정규분포 왜도=0, 첨도=0(초과 첨도) 또는 3(일반 첨도) — 공식에서 -3의 여부로 구분
  • 정규분포 곡선: 좌우대칭, 하나의 꼭지, X축에 닿지 않음

확률 기초 · 사건의 종류

사건의 종류: 독립·배반·종속·근원

★★★★★
독립사건배반사건종속사건
핵심 사건 유형▼
  • 독립사건 — A 발생이 B 발생 확률에 영향 없음
    P(B|A)=P(B), P(A|B)=P(A), P(A∩B)=P(A)·P(B)
    P(A∪B) = P(A)+P(B)-P(A)·P(B)
  • 배반사건 — 교집합이 공집합. P(A∩B)=0, P(A∪B)=P(A)+P(B)
  • 종속사건 — 한 사건이 다른 사건에 영향. P(A∩B)=P(A|B)·P(B)
  • 근원사건 — 표본 공간에 원소가 한 개일 때
조건부 확률
P(A|B) = P(A∩B) / P(B), 단 P(B) > 0

베이즈 정리·심슨의 역설·기저율 무시

★★★★★
심슨의 역설기저율 무시베이즈 정리
확률 오류와 이론▼
  • 심슨의 역설 — 부분집단에서는 일관된 경향을 보이지만, 전체를 합치면 그 경향이 역전되는 현상
  • 기저율 무시 — 사전확률(기저율)을 무시하고 새로운 정보만으로 판단하는 오류
  • 베이즈 정리 — 기저율을 반영하기 위해 사용하는 기본 확률 원리
베이즈 정리 공식
P(A|B) = P(B|A) × P(A) / P(B)

확률분포함수·누적분포함수

★★★★☆
확률질량함수확률밀도함수누적분포함수
분포 함수 분류▼
  • 확률질량함수(PMF) — 이산 확률변수에서 특정값에 대한 확률을 계산
  • 확률밀도함수(PDF) — 연속 확률변수에서 특정 구간에 속할 확률을 계산
  • 누적분포함수(CDF) — 확률변수 X가 특정값 x 이하일 확률을 누적하여 계산
확률변수의 분산
Var(X) = E(X²) - {E(X)}²
σ(X) = √Var(X)
Var(aX ± b) = a²·Var(X)
Var(aX ± bY) = a²Var(X) ± b²Var(Y)
확률분포의 종류

이산 확률분포 종류

★★★★★
이항베르누이포아송초기하
이산 분포▼
  • 베르누이 분포 — 한 번의 시행에서 성공(1)/실패(0). E(X)=p, Var(X)=p(1-p)
  • 이항 분포 — 동일 베르누이 시행을 n번 반복했을 때 성공 횟수. 매 시행 성공확률 같음. E(X)=np, Var(X)=np(1-p)
  • 기하 분포 — 처음 성공이 나올 때까지의 시도 횟수
  • 포아송 분포 — 단위 시간이나 단위 공간(=영역)에 어떤 사건이 발생하는 횟수. E(X)=λ, Var(X)=λ (1개 모수)
  • 초기하 분포 — 유한 모집단에서 비복원추출. 시행들이 서로 독립적이지 않고 성공확률이 매번 일정하지 않음
  • 이산균등분포 — 정해진 정수 범위 내에서 모든 값이 동일한 확률

연속 확률분포 종류

★★★★★
정규분포T분포카이제곱F분포
연속 분포▼
  • 정규분포 — 기댓값=최빈값=중앙값=평균. 왜도=0, 첨도=0(초과)/3(일반). 좌우대칭. X축에 닿지 않음
  • T-분포 — 표본수가 적을 때 평균 비교. 정규분포보다 꼬리가 두꺼움
  • 카이제곱 분포 — 표준정규분포를 만족하는 확률변수 여러 개의 제곱합은 자유도 n인 카이제곱 분포를 따름
  • F-분포 — 두 카이제곱 분포의 비율. 분산 비교, ANOVA
  • 연속균등분포 — 구간 [a, b]에서 모든 값이 동일한 확률
  • 지수분포 — 사건이 일어날 때까지의 대기시간 분포 (오른쪽 꼬리)
  • 감마분포 — 어떤 사건이 여러 번(α번) 일어날 때까지 걸리는 시간
데이터 전처리 · 결측값 처리

데이터 전처리 개요

★★★★☆
70~80% 소모반복적 과정
전처리 특징▼
  • 오류 방지, 신뢰도 향상, 반복적인 과정
  • 실제 분석 과정의 70~80% 소모
  • 전처리 작업 종류: 데이터 정제, 데이터 통합, 데이터 축소, 데이터 변환, 불균형 데이터 처리
  • 데이터 정제: 결측값, 이상값 처리, 중복 제거, 데이터 형식 변환, 공백/특수문자 제거. 반정형/비정형에 노이즈/이상값이 많이 발생

결측값 처리: 단순 대치법

★★★★★
완전사례평균 대치단순확률
단순 대치 3가지▼
  • 완전 사례 분석법 — 결측값이 포함된 관측치를 모두 제외
  • 평균 대치법 — 조건부/비조건부. 표준오차(σ/√n)의 과소추정 문제 발생
  • 단순 확률 대치법 — Hot-Deck(같은 데이터셋에서 유사한 값으로 대치), Cold-Deck(외부 데이터 활용)

결측값 처리: 고급 대치법

★★★★★
모형 기반다중 대치
고급 대치 방법▼
  • 모형 기반 대치법 — 상관도가 높아지고 분산이 작아짐
  • 다중 대치법 — 절차 복잡성, 추정량 계산의 복잡성, 분석 결과 결합의 난해함

📌 결측값 상황별 처리 기준

범주형 변수 → 최빈값
연속형 변수 → 평균
왜도 있을 때 → 중앙값
높은 결측률 + 낮은 중요도 → 변수 제거
높은 결측률 + 높은 중요도 → 회귀 대치 / k-NN 기반 대치 / 다중 대치법

이상값 · 변수 변환 · 부호화

이상값(Outlier) 검출 방법

★★★★★
ESDIQR 방법박스플롯
이상값 검출▼
  • 이상값 특징: 반정형/비정형에서 더 자주 발생. 분포의 집중경향치를 왜곡
  • 이상값 처리: 삭제, 대체(양의 왜도일 때 중앙값으로 대체), 변환, 절단
이상값 검출 공식
ESD: μ - 3σ < 정상데이터 < μ + 3σ
IQR: Q1 - 1.5×IQR < 정상 < Q3 + 1.5×IQR

📦 박스플롯(Boxplot) 포함 요소

Min, Max, Median, Mean, Q1, Q3, IQR=Q3-Q1
lower = Q1 - 1.5×IQR, upper = Q3 + 1.5×IQR
포함하지 않는 것: 평균, 분산, 데이터 개수

스케일링(Scaling) 방법

★★★★★
MinMaxZ-ScoreRobust
정규화·표준화▼
  • Min-Max Normalization — [0~1] 범위로 변환
  • Standardization (Z-Score) — 평균 0, 분산 1의 표준정규분포로 변환. (X-μ)/σ. 표준화된 값은 단위 없음(단위 약분됨)
  • Robust Scaler — IQR 사용, 이상값 영향 최소화
  • Max Absolute Scaler
Min-Max 공식
(X - Xmin) / (Xmax - Xmin)
Z-Score 공식
(X - μ) / σ

비선형→선형 변환▼

  • 왼쪽 꼬리가 길 때(음의 왜도): 제곱, 지수 변환
  • BoxCox 변환 — 정규분포에 가깝게 만들거나 분산 안정화. 양수 데이터에만 적용 가능

부호화(Encoding) 방법

★★★★★
LabelOne-HotBinaryTarget
범주형 인코딩▼
  • Label Encoding — 각 범주를 숫자에 1:1로 대치
  • One-Hot Encoding — 범주의 개수만큼 더미변수 생성 (R에서는 범주 수 - 1). 서로 다른 단어의 내적은 0. 공간 차원 효율이 좋지 않음(Sparse 데이터)
  • Binary Encoding — One-Hot보다 모델 학습 속도가 더 빠름
  • Frequency Encoding — 범주형 변수를 해당 범주의 빈도로 변환
  • (Mean) Target Encoding — 종속변수 값들의 평균을 활용

★★★★☆

변수 종류 비교▼
  • 파생변수 — 기존 데이터 변수들을 가공·연산하여 새로운 의미의 변수 생성. 매우 주관적일 수 있으므로 논리적 타당성 필요. 원본 데이터 기반으로 새로 생성되어 지속적으로 분석에 사용
  • 요약변수 — 여러 관측값 또는 범주를 집계(합계, 평균, 최소, 최대, 개수)하여 집합적 정보를 나타내는 변수. 그룹별 집계 통계량. 데이터 마트에서 중요. 많은 모델에서 공통 사용 가능하여 재활용 가능. 특정 시점(일시적)의 특성을 요약

  • 임베디드 기법 — Lasso, Ridge, ElasticNet, 의사결정나무, 랜덤포레스트(변수의 중요도 계산)

상관계수 비교▼
  • 피어슨 상관계수 — 등간/비율척도. 선형적인 관계만 측정. 이상값의 영향 받음. 통계적 유의성 판단 불가. = X,Y의 공분산 / X,Y의 표준편차의 곱

  • 포인트 바이세리얼 — 하나의 변수는 이분형, 다른 하나는 연속형

대수의 법칙·중심극한정리

★★★★★
대수의 법칙중심극한정리 n≥30
표본의 핵심 정리▼
  • 대수의 법칙 — 표본의 크기(n)가 커질수록 표본평균은 모집단의 평균에 더 가까워진다
  • 중심극한정리 — 모집단의 분포와 상관없이 표본의 크기(n)가 30 이상이 되면 N이 커짐에 따라 표본평균의 분포가 정규분포에 근사해짐
표본평균의 분포
평균: μ, 분산: σ²/n, 표준편차(표준오차): σ/√n
표본오차 = 오차한계 = 임곗값 × σ/√n

★★★☆☆

EDA 개요▼

  • 교차 분석: 명목형/서열형 변수 간의 분포와 관계를 확인. 질적 자료 간의 연관성 탐색. 독립성 검정(카이제곱 검정)에 사용

    • 포아송분포 최대우도추정량: x̄

구간추정·신뢰구간

★★★★★
신뢰구간오차한계표준오차
구간추정 핵심▼
  • 구간추정: 신뢰구간을 만들어 추정. 점추정 값이 먼저 필요함
  • 신뢰구간: (통계량 - 오차한계) ~ (통계량 + 오차한계)
오차한계 = 표본오차
오차한계 = 임곗값 × 표준오차 = 임곗값 × σ/√n
  • 신뢰수준이 높을수록 신뢰구간이 길어짐: 99% 신뢰구간 > 95% 신뢰구간
  • 표본의 크기가 커지면 신뢰구간의 길이는 줄어든다

가설검정: 귀무가설·대립가설

★★★★★
귀무가설대립가설p-value기각역
가설검정 핵심▼
  • 귀무가설(H₀) — 가설검정 비교의 기준. 효과 없음, 변화 없음, 차이 없음. 유의한 차이가 없을 때 채택
  • 대립가설(H₁) — 귀무가설이 기각되면 채택. 연구자가 검증하고자 하는 주장. 채택되면 통계적으로 유의한 차이 있음
  • 제1종 오류(α) — 귀무가설이 참인데 기각되는 오류
  • 제2종 오류(β) — 귀무가설이 거짓인데 채택되는 오류
  • p-value: 제1종 오류를 범할 확률, 귀무가설을 지지하는 정도. p-value ≤ α이거나 검정통계량 ≥ 임곗값일 때, 귀무가설 기각, 대립가설 채택
가설검정 절차
가설 설정 → 유의수준 설정 → 검정통계량 산출(p-value 계산) → 기각/채택 판단

검정통계량 공식 정리

★★★★☆
Z-분포t-분포카이제곱F-분포
검정통계량 공식▼
Z-검정 (모분산 알 때)
Z = (x̄ - μ₀) / (σ/√n)
t-검정 (모분산 모를 때, 소표본)
t = (x̄ - μ₀) / (s/√n)
카이제곱 검정 (분산 검정)
χ² = (n-1)s² / σ₀²
F-검정 (두 분산 비교)
F = s₁² / s₂²
  • 양측검정, 좌측검정, 우측검정: 기각역의 위치에 따라 구분
