머신러닝 개요 · 학습 유형

지도학습 vs 비지도학습 vs 준지도학습

★★★★★
지도학습비지도학습준지도학습
핵심 분류▼
  • 지도학습 — 모든 데이터에 정답(레이블)이 필요
    • 회귀: 연속형 자료 예측 (집값, 선형 회귀, Lasso, Ridge, ElasticNet)
    • 분류: 범주형 자료 예측 (손글씨 숫자, 이메일 스팸, 로지스틱 회귀)
    • 회귀·분류 모두: 인공신경망(MLP), 의사결정나무, SVM, k-NN, 앙상블
  • 비지도학습 — 정답(레이블) 없이 데이터만으로 구조·패턴 학습
    • 군집: K-means, DBSCAN, SOM
    • 연관: 상품 구매 연관성 분석, Apriori, FP-Growth
    • 차원 축소: 주성분 분석, 오토인코더
  • 준지도학습 — 정답이 표시된 데이터 + 표시되지 않은 데이터를 모두 훈련에 사용

딥러닝 유형

★★★★★
CNNGANRNN강화학습
딥러닝 종류▼
  • CNN — 이미지 처리. 예: 얼굴 인식 보안 게이트
  • GAN — 이미지 생성. 예: 피카소 화풍 그림 생성
  • RNN — 자연어 처리. 예: 영화 감상평 긍정/부정 판단
  • 강화학습 — 인공지능이 환경에서 특정 목표 달성을 위해 시행착오를 통해 스스로 학습. 예: 공장 로봇 훈련
  • 전이학습 — 미리 학습된 모델의 지식을 새로운 문제에 적용하는 학습 기법

분석모형 구축 절차

★★★★☆
분석기획모델링운영화
구축 절차▼
  • 분석모형 구축 절차: 분석 요건 정의 → 모델링 → 검증 및 테스트 → 운영화 및 적용
  • 기계학습 모델 구축 절차: 과제 정의 → 데이터 수집 → EDA 및 데이터 전처리 → 모델 학습 → 모델 성능평가
  • 데이터 분석 모델링 절차: 모델링 마트 설계 → 탐색적 분석 → 모델링 → 성능평가 → 운영 테스트
  • 주의: 설명력이 매우 높은 모형은 예측력이 떨어질 수 있음(Overfitting)

홀드아웃(Hold-out) vs K-Fold 교차검증

★★★★★
홀드아웃K-FoldLOOCV
검증 방법 비교▼
  • 홀드아웃: 2개(훈련·평가) 또는 3개(훈련·검증·평가)로 분리. 랜덤 추출. 훈련과 평가 데이터는 겹칠 수 없음. 검증 데이터: 과대적합 여부 확인, Early Stopping, 하이퍼파라미터 조정에 사용
  • K-Fold: Shuffle 후 K개 동일 크기 셋, (K-1)개 학습셋 + 1개 검증셋으로 K번 반복. K-Fold는 홀드아웃보다 느림. 과대적합 방지. 분류: Stratified K-Fold 사용. 데이터 충분하지 않을 때 사용. 시계열 데이터: 시간순으로 나누어 검증
  • LOOCV: K=n인 교차검증. 데이터 수가 적을 때 사용. 두 분석가가 동일한 결과를 얻을 수 있음

관측치를 한 번 이상 훈련용으로 사용하는 복원추출법 기반 평가 방법
특정 표본이 뽑히지 않을 확률: d = (1 - 1/N)ⁿ

회귀분석

회귀모형의 종류

★★★★★
단순선형다중선형다항로지스틱
회귀 유형▼
  • 단순 선형 회귀 — 독립변수 1개, 종속변수와 1차 선형 관계
  • 다중 선형 회귀 — 독립변수 2개 이상, 1차 선형 관계. 잔차의 자유도 = 표본 크기 – 독립변수 수 – 1
  • 다항 회귀 — 독립변수 1개 이상, 종속변수와 2차 이상(곡선) 관계
  • 로지스틱 회귀 — 종속변수가 범주형인 경우

⚠️ 시험 포인트

변수 조합이 달라지거나 조건이 달라지면 회귀계수가 달라짐
→ 다중회귀 결과를 보고 단순회귀 결과를 알 수 없음

회귀모형의 5가지 가정

★★★★★
선형성독립성정규성등분산성
회귀 가정▼
  • 선형성 — 독립변수의 변화에 따라 종속변수도 변화하는 선형모형
  • 독립성 — 잔차와 독립변수의 값이 관련되어 있지 않음
  • 정규성(Normality) — 잔차항이 정규분포를 이뤄야 함
  • 등분산성 — 독립변수의 모든 값에 대해 잔차항들의 분포는 동일한 분산
  • 비상관성 — 관측치들의 잔차들끼리 상관이 없어야 함

📊 모델 진단 그래프 해석

Normal Q-Q: 정규성 / Scale-Location: 등분산성
Residuals vs Leverage: 영향점 / Residuals vs Fitted: 선형성·등분산성

회귀모형 유의성 검정

★★★★★
F통계량결정계수 R²p-value
유의성 검정▼
  • 모형 유의성: F 통계량 사용. p-value ≤ 유의수준 → 귀무가설 기각 → 유의미
  • 회귀계수 유의성: 회귀계수의 t값. p-value ≤ 유의수준 → 유의미
  • 결정계수(R²) = SSR/SST = 1 - (SSE/SST). 0~1 사이, 1에 가까울수록 높은 설명력. 독립변수가 많아지면 커짐
  • 절편: 독립변수가 모두 0일 때 종속변수의 예상값
  • 기울기 회귀계수: 양수이면 종속변수 증가, 음수이면 감소

📋 ANOVA 분산분석표 (p=절편 제외 회귀계수 수)

Regression: df=p, Sum Sq=SSR, Mean Sq=MSR=SSR/p, F=MSR/MSE
Residuals: df=N-p-1, Sum Sq=SSE, Mean Sq=MSE=SSE/(N-p-1)

정규성 검정 기법

★★★☆☆
Shapiro-WilkK-S TestQ-Q Plot
정규성 검정▼
  • 정규성 검정 기법: Normal Q-Q Plot, Histogram, Shapiro-Wilk Test, Kolmogorov-Smirnov Test, Anderson-Darling Test
  • 과대적합: 일반화 성능이 낮은 상태. 비선형 모형에서 더 쉽게 발생. 훈련 성능이 평가 성능보다 많이 낮은 상태
벌점화 회귀(규제 모형) · 로지스틱 회귀

Lasso vs Ridge vs ElasticNet

★★★★★
Lasso(L1)Ridge(L2)ElasticNet
벌점화 회귀 비교▼
  • 벌점화 회귀: 회귀계수의 크기에 벌점(Penalty)을 부여하여 계수의 변동폭(분산)을 줄이고, 약간의 편향(Bias)을 허용하여 과대적합 방지
  • Lasso(L1 norm) — 절댓값 사용. 회귀계수가 0이 되거나 0에 가깝게 됨 → 변수 선택 효과
  • Ridge(L2 norm) — 제곱 사용. 회귀계수가 0에 가까워질 뿐 0은 되지 않음
  • 람다(λ)값이 클수록 패널티를 강하게 준 것
  • 다중공선성 문제를 완화하는 데 활용 가능

로지스틱 회귀 모형

★★★★★
sigmoidoddslogitMLE
로지스틱 회귀 핵심▼
  • 로지스틱 회귀: 지도학습, 이항분류 및 다항분류에 사용. 종속변수: 범주형(이산형), 이항분포. 0~1 출력. 대표적인 일반화 선형모형
  • 추정 방법: 최대우도법(MLE), 가중최소자승법(WLS), 카이제곱 검정
  • 승산(odds) = 성공확률/실패확률 (0 < odds < ∞)
  • logit = log(odds) = z = β₀ + β₁x₁, (-∞ < logit < ∞). x₁이 1단위 증가하면 odds는 e^β₁만큼 증가
  • 승산비(odds ratio): 1보다 크면 odds 증가, 1보다 작으면 odds 감소
sigmoid 함수 (0~1 변환)
P(Y=1) = 1 / (1 + e⁻ᶻ)

일반화 선형모형 · 더미변수

★★★★☆
링크함수더미변수교호작용항
일반화 선형모형▼
  • 일반화 선형모형(GLM): 종속변수가 정규분포를 따르지 않는 경우에도 사용 가능한 확장된 선형모형. 3요소: 확률분포, 선형예측자, 링크함수
  • 링크함수: 포아송분포→log함수, 이항분포→로짓, 정규분포→항등함수
  • 더미변수: 범주형 변수를 0과 1로 변환. gender=0이면 y=β₀, gender=1이면 y=β₀+β₁
  • 교호작용항: 한 변수의 효과가 다른 변수의 수준에 따라 달라지는 관계. 교호작용항이 유의하다 = 두 변수의 효과가 독립적이지 않다
  • 유의하지 않더라도 회귀식에서 제거하지 않음
의사결정나무 · SVM · k-NN

의사결정나무 특징과 결정규칙

★★★★★
비모수분할규칙가지치기과대적합
의사결정나무 핵심▼
  • 특징: 분류·예측 모두 가능. 비모수 방법(정규성 가정 불필요). 해석 용이. 수치형·범주형 모두 사용. 과대적합 우려. 주요 알고리즘: CART, C5.0, C4.5, CHAID
  • 분할규칙: 부모마디보다 자식마디의 순수도가 증가하도록 분기 (불순도 감소, 동질성 증가). 자식 노드에 한 가지 종류의 클래스가 존재할 때까지 분리
  • 정지규칙: 깊이 최대이면 멈춤, 마지막 가지 개수가 일정 미만이면 멈춤, 불순도 개선 없으면 멈춤. 뿌리 노드만 남는 이유: 변별력 있는 변수가 없어서
  • 가지치기규칙: 복잡한 트리 간단히 → 과대적합 방지, 일반화 성능 향상. 비용 함수 최소화

📊 분리 기준

분류나무: 지니 지수, 엔트로피 지수, 카이제곱 통계량(p-value)
지니 지수 = 1 - Σ(각 범주별 데이터 수/전체 데이터 수)²
회귀나무: F-통계량(p-value), 분산 감소량

SVM(Support Vector Machine)

★★★★★
커널 트릭소프트 마진과적합
SVM 핵심▼
  • SVM: 지도학습. 이진분류, 다항분류, 회귀(SVR) 모두 사용. 설명력이 부족함
  • 노이즈 많은 데이터에 과적합 위험. 속도가 느린 편
  • 커널 트릭: 비선형 데이터셋 분류를 위한 고차원 특징 공간으로 사상
  • 소프트 마진: 일부 오차를 허용하는 유연한 경계 설정
  • SVM에서 가중치 벡터는 초평면과 직교, 편향은 offset 결정
  • RBF(가우시안) 커널: 하이퍼파라미터 C, gamma. gamma 값이 클수록 과적합 위험

k-NN (k-Nearest Neighbors)

★★★★☆
lazy learninghyperparameter k과대적합
k-NN 특징▼
  • k-NN: lazy learning(인스턴스 러닝). 새로운 데이터에 대해 이웃 k개의 가장 가까운 멤버들과 비교하여 결과 판단
  • 이상값의 영향: K가 작을 때 더 크게 나타남 → 과대적합은 K가 작을 때 심함
  • 스케일링이 중요. k: 하이퍼파라미터
앙상블 모형

앙상블 종류: Bagging vs Boosting vs Stacking

★★★★★
Bagging(분산감소)Boosting(편향감소)Stacking
앙상블 유형 비교▼
  • Voting: 동일 데이터셋, 서로 다른 여러 알고리즘. Hard voting(빈도수), Soft voting(확률)
  • Bagging(분산 감소): 동일 알고리즘, 복원추출(부트스트랩) 표본에 병렬로 학습, 결과 평균(다수결). 서로 다른 데이터셋, 동일 알고리즘. 모델: Random Forest, Extra Trees, Bagged SVM, Bagged k-NN
  • Pasting: Bagging과 비슷하지만 비복원추출 방식
  • Boosting(편향 감소): 여러 모델 순차적 학습. 잘못 분류된 데이터에 가중치 부여. 이상치에 약함. 이전 학습 결과를 반영(약학습기→강학습기). 모델: AdaBoost, Gradient Boost, XGBoost, LightGBM, CatBoost
  • Stacking: 서로 다른 여러 알고리즘 모델들의 예측 결과를 새로운 데이터로 사용해 메타 모델을 학습시켜 최종 결과 도출

⚠️ 앙상블 특징

여러 분류 모형 결합 → 정확도 향상. 해석력은 낮음.
단일 모델보다 성능 높지만 "항상" 좋은 것은 아님.
과적합 감소 효과(일반화된 모형)

인공신경망(ANN) · 딥러닝

인공신경망(ANN) 구조와 학습

★★★★★
가중치역전파기울기 소실
ANN 핵심▼
  • ANN: 분류·예측 모두 가능. 입력층·은닉층·출력층 구성. 각 층 완전 연결
  • MLP 가중치 수 = (입력노드×은닉노드) + (은닉노드×출력노드)
  • 단층퍼셉트론(SLP): OR, AND, NOR 학습 가능 / MLP: XOR 학습 가능

연관규칙 측정지표: 지지도·신뢰도·향상도

★★★★★
지지도신뢰도향상도
연관규칙 핵심▼
  • 연관분석: 항목들 간 '조건-결과' 패턴 발견. 장바구니 분석. 비목적성 분석
지지도(Support)
P(A∩B) = A와 B 동시 포함 거래 수 / 전체 거래 수
신뢰도(Confidence, A→B)
P(B|A) = P(A∩B) / P(A) = A·B 동시 포함 / A 포함 거래 수
향상도(Lift)
P(B|A) / P(B) = P(A∩B) / (P(A)·P(B))
  • 향상도 > 1: 연관성 높음, 양의 관계
  • 향상도 = 1: 두 품목 사이 상호 관계 없음
  • 향상도 < 1: 음의 상관관계

시계열 정상성(Stationarity)

★★★★★
정상성차분백색잡음
정상성 개념▼
  • 정상성 조건:
    • 평균은 모든 시점 t에 대해 일정: E(xₜ) = μ
    • 분산은 모든 시점 t에 대해 일정: Var(xₜ) = σ²
    • 공분산은 시점에 의존하지 않고 시차에만 의존: Cov(xₜ₊ₕ, xₜ) = γₕ
  • 정상 시계열 전환:
    • 평균 일정하지 않은 경우 → 차분
    • 계절성이 있는 경우 → 계절 차분
    • 분산이 일정하지 않은 경우 → 자연 로그(변환)
  • 백색잡음(White Noise): 자기상관이 전혀 없는 경우. 서로 독립. 평균=0, 분산 일정, 자기공분산=0. 정상성 만족

★★★★★

시계열 모형▼
  • AR 모형: 백색 잡음의 현재 값과 자기 자신의 과거 값의 선형 가중 값. 정상 확률 모형(조건부 정상 시계열). ACF가 지수적 감소
  • MA 모형: 현시점 자료가 유한 개의 과거 백색잡음의 선형결합. 정상 시계열. PACF가 지수적 감소
  • ARIMA 모형: 비정상시계열 모형. 차분/변환을 통해 AR, MA, ARMA 모형으로 정상화. 예: ARIMA(1,2,3) → 2번 차분 → ARMA(1,3)

  • Bag of Words: 단어 순서를 고려하지 않고 출현 빈도에만 집중
  • TF-IDF: 어떤 단어가 특정 문서 내에서 얼마나 중요한지를 나타내는 통계적 수치
  • Word Embedding: 단어의 의미를 실수값의 집합인 밀집 벡터 형태로 표현. 유사도 값이 클수록 두 벡터 간 유사도가 더 높음. 가까운 거리 = 의미적으로 유사
  • Word2Vec: Word Embedding의 대표적인 방법 중 하나
